游戏规则

基本规则

  • 四人游戏，三打一（1名庄家 vs 3名闲家）
  • 使用两副牌，去掉3和4，共92张牌
  • 留8张底牌，每人发21张牌
  • 王、2、7为常主

叫分抢庄

  • 从100分起叫，每次最少降5分，最低叫到75分
  • 叫分最低者为庄家

计分

  • 5、10、K为分牌
  • 闲家得分 ≥ 庄分：庄家下庄
  • 抠底可获得底牌分数加成
