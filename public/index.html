游戏规则

基础

  • 四人游戏，一名庄家对抗三名闲家。
  • 两副牌共 108 张，每人 25 张，底牌 8 张。
  • 5、10、K 为分牌，闲家抢分，庄家守分。

常主

  • 王、2、7 为常主，无论叫何花色均属主牌。
  • 大小以花色区分：副2 < 主2 < 副7 < 主7 < 小王 < 大王。

叫分

  • 叫分从 100 开始，按 5 分递减，最低 75。
  • 叫分最低者为庄家；直接叫 75 立即坐庄。
  • 全员不叫则比常主数量，最多者判负。

出牌

  • 首家可出单张、对子、拖拉机或同花色组合。
  • 跟牌必须同张数；有首家花色时必须尽量跟足。
  • 有对子或拖拉机时，需按牌型义务跟出。

结算

  • 闲家得分 ≥ 庄分：庄家下庄。
  • 闲家以主牌赢得最后一轮可抠底，底牌分按倍数计入。
  • 清光（闲家 0 分）或边光（<30 分）时庄家结算翻倍。
